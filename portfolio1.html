Portfolio de Projetos de Luiz Filipe

"Gerador de Insultos Educados" - Aplicativo de Diversão Social:

  • Descrição: Um aplicativo que gera insultos de forma educada e humorística. Os usuários podem inserir o nome da pessoa que desejam "elogiar" e o aplicativo gera um insulto leve e engraçado, mas inofensivo.
  • Tecnologias Utilizadas: Vue.js (para frontend interativo), Flask (para backend simples), banco de dados para armazenar insultos e nomes personalizados.
  • Toque Engraçado: Os insultos são completamente inofensivos e, de fato, engraçados. Tem as opções para compartilhar os insultos nas redes sociais, tornando-os uma forma única de entretenimento entre amigos.


"Tradutor de Caninos para Humanos" - Aplicativo de Comunicação Interespécie:

  • Descrição: É um aplicativo que "traduz" os latidos dos cães para frases compreensíveis pelos humanos. Os usuários podem gravar os latidos de seus cães e o aplicativo os "traduz" para mensagens engraçadas ou atitudes caninas.
  • Tecnologias Utilizadas: React Native (para desenvolvimento móvel multiplataforma), algoritmos de processamento de áudio simples para análise dos latidos.
  • Toque Engraçado: Cria frases absurdas e divertidas que os cães supostamente estariam dizendo, como "Comida! Comida! Comida!" ou "Vamos dar um passeio agora!". Tambem tem opções para compartilhar as traduções nas redes sociais, aumentando a diversão entre os donos de cães.


"Simulador de Reuniões de Família Caóticas" - Jogo Interativo:

  • Descrição: Crie um jogo de simulação onde os jogadores devem lidar com situações engraçadas e caóticas que acontecem durante reuniões de família. Desde tias fofoqueiras até primos atrapalhados, os jogadores devem tomar decisões para manter a paz e evitar desastres.
  • Tecnologias Utilizadas: Unity (para desenvolvimento de jogos), C# para a lógica do jogo, arte e animações caricatas para os personagens.
  • Toque Engraçado: Encha o jogo com humor familiar, como piadas internas da família e situações exageradas que todos podem relacionar, como a briga pela última fatia de bolo ou o cachorro que rouba os petiscos da mesa.
